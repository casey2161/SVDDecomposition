To run the program in windows
	Open up the command prompt and navigate to the location of SVDdecomp.jar
	Type java -jar SVDdecomp.jar
To run the program in *nix
	Open up terminal and navigate to the location of SVDdecomp.jar
	Type java -jar SVDdecomp.jar
